## API

Calendar的属性说明如下：

| 属性 | 说明 | 可选值 | 类型 | 默认值 | 是否必传 |
| --- | --- | --- | --- | --- | --- |
| firstDayOfWeek | 每周第一天为哪天 | 1/2/3/4/5/6/7 | Number | 1 | false |

示例：
``` js
<Button size="small" type="primary" :isFill="false">急速打卡</Button>
```
