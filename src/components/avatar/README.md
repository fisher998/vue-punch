## API

Avatar的属性说明如下：

| 属性 | 说明 | 类型 | 默认值 | 是否必传 |
| --- | --- | --- | --- | --- |
| src | 图片类头像的资源地址 | String | - | false |
| click | 点击头像click事件 | function | - | false |

示例：
``` js
<Avatar src="..." @click="handleClick"></Avatar>
```
