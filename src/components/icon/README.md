## 何时使用

icon

## API

Avatar的属性说明如下：

| 属性 | 说明 | 可选值 | 类型 | 默认值 | 是否必传 |
| --- | --- | --- | --- | --- | --- |
| type | icon类型 | - | String | - | false |
| size | icon大小,px | - | String/Number | - | false |
| color | icon颜色 | - | String | - | false |

示例：
``` js
 <Icon type="sdf" size="100" color="red"></Icon>
```
